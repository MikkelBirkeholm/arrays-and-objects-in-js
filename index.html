Arrays and objects

Arrays

let myArray = ['Tekst', 80085, 'Tekst2']

Du kan vise data fra et array ved at referere til en indeks i det pågældende array. F.eks. "myArray[0]"

Objects

let myObject = {
        firstName: 'Johnny',
        lastName: 'Cash',
        occupation: 'Musician',
        url: 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
        ringOfFire: function () {
            window.open(this.url)
        }
    }

Du kan vise data fra et objekt ved at referere til objektet samt en key. F.eks. "myObject.firstName"
